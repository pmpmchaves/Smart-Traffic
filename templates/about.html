Brief description

Traffic congestion has become a major issue all around the world, thus policymakers and city managers have huge challenges to improve the quality of life in urban areas. It is a widely occurring traffic phenomenon in metropolitan areas due to the increasing number of road vehicle users and is characterized by slower vehiclesâ€™ speeds, longer travel times, queues and, sometimes, complete paralysis of the traffic network. Smart-Traffic is Mobility as a Service application for real-time traffic prediction.

This proof of concept relies on the work of Dr. Taiwo O. Adetiloye:

Adetiloye, Taiwo Olubunmi (2018), Predicting Short-Term Traffic Congestion on Urban Motorway Networks. Ph.D. thesis, https://spectrum.library.concordia.ca/985249/, Concordia University, Montreal, QC, Canada

The following three-steps are required for setup:

  • 1. Install Docker

  • 2. Execute the following docker commands from the command line:

    • docker pull taiwotman/smart-traffic:beta1.1
    docker run --rm -p <desired-port>:5000 <image-id>

    For example:

    docker pull taiwotman/smart-traffic:latest docker run --rm -p 80:5000 taiwotman/smart-traffic:latest

  • 3. On your browser, open: http://localhost:80

Note:

Due to latency, the API request takes an average of 1min.

Kindly, exercise some patience to complete page load with predictions after the submit button is pressed.

Sample codes on Github repository: https://github.com/taiwotman/TensorflowPredictCongestionTypes
